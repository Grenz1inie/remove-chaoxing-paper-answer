# 更新日志 (CHANGELOG)

## [2.0.0] - 2025-11-08

### ✨ 重大更新：模块化重构

#### 新增
- ✅ 采用现代化模块化设计，代码按功能拆分为独立模块
- ✅ 新增 `modules/config.js` - 配置管理模块
- ✅ 新增 `modules/dom-handler.js` - DOM操作封装模块
- ✅ 新增 `modules/ui.js` - UI界面管理模块
- ✅ 新增 `modules/button-controller.js` - 按钮控制逻辑模块
- ✅ 新增 `MODULE_DESIGN.md` - 详细的模块化设计文档
- ✅ 通过 `@require` 实现模块自动加载

#### 改进
- 🔧 重构主入口文件 `main.user.js`，代码更简洁清晰
- 🔧 配置项集中管理，修改更方便
- 🔧 函数功能单一化，遵循单一职责原则
- 🔧 添加详细的代码注释和JSDoc文档
- 🔧 统一日志输出格式
- 🔧 改进错误处理机制

#### 优化
- ⚡ 代码可读性大幅提升
- ⚡ 代码可维护性显著增强
- ⚡ 便于后续功能扩展
- ⚡ 模块可独立测试和复用

#### 技术细节
- 📦 模块化架构：5个独立文件，职责清晰
- 🔗 依赖管理：通过 `@require` 声明模块依赖
- 🌐 部署方式：支持 GitHub Raw URL 和 CDN 加载
- 📝 文档完善：新增详细的架构和使用说明

## 版本规范说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号（Major）**：进行不兼容的 API 修改
- **次版本号（Minor）**：向下兼容的功能性新增
- **修订号（Patch）**：向下兼容的问题修正
